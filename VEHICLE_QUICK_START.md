# 车辆数据录入 - 快速上手指南

## 🚀 3步快速录入车辆

### 第1步：进入车辆管理页面

**方法1：通过侧边栏**
```
登录系统 → 左侧侧边栏 → 点击"车辆管理"（🚗图标）
```

**方法2：直接访问**
```
浏览器地址栏输入：/vehicles
```

---

### 第2步：点击"车辆入库"按钮

在车辆管理页面：
```
页面右上角 → 蓝色按钮"+ 车辆入库"
```

---

### 第3步：填写车辆信息并保存

弹出的表单对话框包含以下部分：

#### ✅ 必填信息（最少录入）
```
1. 车架号后六位：______ （6位字符）
2. 车牌号：______
3. 品牌：______ （如：大众）
4. 型号：______ （如：帕萨特）
5. 年份：______ （如：2020）
6. 里程数：______ （如：50000）
7. 过户次数：______ （如：1）
8. 购车款：______ （如：150000）
9. 选择出资人：☑ 押车出资人、☑ 场地老板
```

#### 📝 可选信息（推荐填写）

**基本识别信息**
- 完整车架号（VIN）：17位
- 发动机号

**车辆类型**
- 下拉选择：轿车 / SUV / MPV / 跑车 / 两厢车 / 皮卡 / 面包车 / 运动型

**技术参数**
- 排量（升）：如 1.5、2.0
- 变速箱类型：手动 / 自动 / CVT / 双离合 / AMT
- 驱动方式：前驱 / 后驱 / 全时四驱 / 分时四驱
- 燃料类型：汽油 / 柴油 / 纯电动 / 混合动力 / 插电混动
- 排放标准：国三 / 国四 / 国五 / 国六A / 国六B
- 座位数：2座 / 4座 / 5座 / 6座 / 7座 / 8座 / 9座

**外观信息**
- 车身颜色：白色 / 黑色 / 银色 / 灰色 / 红色 / 蓝色 / 金色 / 棕色 / 绿色 / 黄色 / 橙色 / 紫色 / 其他
- 内饰颜色：黑色 / 米色 / 棕色 / 灰色 / 红色 / 白色 / 其他

**车辆状态**
- ☑ 事故车（如有）
- ☑ 泡水车（如有）
- ☑ 火烧车（如有）
- 车况描述：详细说明车辆状况

**证件信息**
- 保险到期日：选择日期
- 年检到期日：选择日期

**价格信息**
- 新车指导价：如 200000
- 购车款：如 150000（必填）

**车辆照片**
- 点击上传或拖拽照片
- 最多10张

#### 💾 保存
```
填写完成 → 点击底部"保存"按钮
```

## ⚡ 快捷操作

### 键盘快捷键
- **Tab**：切换到下一个字段
- **Shift + Tab**：切换到上一个字段
- **Enter**：在最后一个字段按Enter提交表单

### 鼠标操作
- **点击标签**：自动聚焦到对应输入框
- **拖拽照片**：直接拖拽图片文件到上传区域

## ✅ 录入检查清单

录入完成后，请检查：

- [ ] 车架号后六位是否正确（6位）
- [ ] 车牌号是否正确
- [ ] 品牌型号是否准确
- [ ] 年份、里程数、过户次数是否合理
- [ ] 购车款是否正确
- [ ] 出资人是否选择正确
- [ ] 车况描述是否详细
- [ ] 照片是否清晰完整
- [ ] 保险和年检日期是否准确

## 🔍 查看已录入车辆

### 在车辆列表中
```
车辆管理页面 → 车辆列表 → 找到车辆 → 点击"查看"按钮
```

### 查看内容
- 所有录入的信息
- 车辆照片
- 出资人信息
- 成本明细（如有）
- 销售记录（如有）

## 💡 温馨提示

1. **录入前准备**：准备好行驶证、车辆照片、购车发票等资料
2. **分步录入**：可以先录入必填项保存，后续再补充完整信息
3. **及时保存**：填写过程中建议及时保存，避免数据丢失
4. **照片质量**：上传清晰的照片，有助于车辆展示和销售
5. **如实填写**：特别是事故车、泡水车等信息，务必如实填写
