## 🧪 测试建议

### 测试数据隔离

1. 退出当前登录
2. 创建测试车行A：
   ```
   车行名称：测试车行A
   车行代码：test-a
   管理员：admin-a / 123456
   ```
3. 在车行A创建一些数据（员工、车辆）
4. 退出登录
5. 创建测试车行B：
   ```
   车行名称：测试车行B
   车行代码：test-b
   管理员：admin-b / 123456
   ```
6. 验证车行B看不到车行A的数据

### 测试员工加入

1. 退出登录
2. 访问注册页面
3. 选择"加入车行"
4. 输入车行代码：yichi（或 test-a）
5. 填写员工信息并注册
6. 验证可以看到对应车行的数据

4. **Super Admin**：需要在数据库中手动设置角色
