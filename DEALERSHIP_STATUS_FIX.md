## 🔄 车行状态流转

### 正常流程
```
1. 车行注册
   ↓
2. status = 'pending'（待审核）
   ↓
3. 平台审核
   ├─ 通过 → status = 'active'（正常运营）
   └─ 拒绝 → status = 'rejected'（已拒绝）
   ↓
4. 运营中
   ├─ 停用 → status = 'inactive'（已停用）
   └─ 继续运营
   ↓
5. 已停用
   └─ 启用 → status = 'active'（正常运营）
```
