# 🚀 一键测试指南

## 第1步：打开登录页面

**在浏览器中访问：**
```
http://localhost:5173/login
```

---

## 第2步：使用超级管理员登录

**输入以下信息：**

| 字段 | 值 |
|------|-----|
| 用户名 | `吴韩` |
| 密码 | 您设置的密码（如果忘记，可以重置） |

**点击"登录"按钮**

---

## 第3步：进入车行管理

**登录成功后：**

1. 查看左侧导航栏
2. 找到 **"车行管理"** 菜单项（只有超级管理员可见）
3. 点击进入

---

## 第4步：审核车行

**在车行管理页面：**

### 查看待审核列表
- 默认显示"待审核"标签页
- 看到红色徽章：🔴 2（表示有2个待审核车行）
- 列表显示：
  - 好淘车（befgsh）
  - 好淘车（benedg）

### 审核第一个车行（通过）
1. 点击第一个车行的 **"查看"** 按钮（眼睛图标）
2. 查看详细信息：
   - 车行名称、代码
   - 联系人、电话
   - 营业执照图片
3. 关闭详情对话框
4. 点击 **"通过"** 按钮（绿色，带 ✓ 图标）
5. 看到成功提示："车行审核通过"
6. 车行自动移至"正常运营"标签页

### 审核第二个车行（拒绝）
1. 点击第二个车行的 **"拒绝"** 按钮（红色，带 ✗ 图标）
2. 在弹出的对话框中填写拒绝原因：
   ```
   营业执照信息不清晰，请重新上传高清图片
   ```
3. 点击 **"确认拒绝"**
4. 看到成功提示："已拒绝车行注册申请"
5. 车行自动移至"已拒绝"标签页

---

## 第5步：验证登录限制

### 测试1：审核通过的车行可以登录
1. 点击右上角用户菜单，选择"退出登录"
2. 返回登录页面
3. 使用"吴韩"账号登录
4. **预期结果**：✅ 成功登录（因为易驰汽车是 active 状态）

### 测试2：待审核车行无法登录
**注意**：由于我们刚才审核了所有待审核车行，现在没有待审核状态的车行了。如果要测试，需要重新提交注册申请。

## ❓ 常见问题

### Q: 忘记密码怎么办？
A: 可以通过数据库重置密码：
```sql
-- 查看用户信息
SELECT id, username, role FROM profiles WHERE username = '吴韩';

-- 如果需要重置密码，需要通过 Supabase Auth 重置
```

### Q: 找不到"车行管理"菜单？
A: 请确认您的账号角色是 `super_admin`。可以通过以下 SQL 查询：
```sql
SELECT username, role FROM profiles WHERE username = '吴韩';
```

### Q: 如何创建新的待审核车行？
A: 访问公共主页 http://localhost:5173/register，点击"注册新车行"按钮，填写信息并提交。
