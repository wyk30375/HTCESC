## 🎨 表格样式说明

### 字段显示规则
- **必填字段**：始终显示
- **可选字段**：未填写时显示 `-`
- **数字格式**：自动添加千分位分隔符（如 50,000）
- **货币格式**：自动添加 `¥` 符号和千分位分隔符（如 ¥150,000）
- **枚举字段**：自动转换为中文显示（如 sedan → 轿车）

### 颜色标识
- **车况正常**：绿色文字 `text-green-600`
- **车况异常**：红色标签 `variant="destructive"`
- **在售状态**：蓝色标签 `variant="default"`
- **已售状态**：灰色标签 `variant="secondary"`

### 响应式设计
- **横向滚动**：表格宽度超出屏幕时，自动启用横向滚动
- **文字不换行**：使用 `whitespace-nowrap` 保持单行显示
- **移动端优化**：在移动设备上可左右滑动查看所有字段

## 📱 使用说明

### 查看车辆列表
1. 登录系统
2. 进入"车辆管理"页面
3. 选择"在售车辆"或"已售车辆"标签页
4. 查看车辆列表

### 横向滚动
- **桌面端**：鼠标滚轮 + Shift 键横向滚动
- **移动端**：手指左右滑动

### 字段排序
表格按以下顺序展示字段：
1. 基本识别信息（车架号、车牌号）
2. 车辆基本信息（品牌型号、类型、年份、里程、过户）
3. 技术参数（排量、变速箱、驱动、燃料、排放、座位）
4. 外观信息（车身颜色、内饰颜色）
5. 车辆状态（车况）
6. 证件信息（保险、年检）
7. 价格信息（新车指导价、购车款）
8. 出资人信息（押车出资人、场地老板）
9. 其他信息（状态、操作）

## 💡 使用建议

### 1. 完整录入信息
- 建议录入所有字段信息，便于查看和管理
- 特别是技术参数和外观信息，有助于客户了解车辆

### 2. 及时更新证件信息
- 定期检查保险到期日和年检到期日
- 避免证件过期影响车辆销售

### 3. 如实标记车况
- 准确标记事故车、泡水车、火烧车
- 避免后期纠纷

### 4. 合理定价
- 参考新车指导价设置购车款
- 考虑车况、里程、年份等因素

## ⚙️ 技术说明

### 数据来源
- 所有字段数据来自 `vehicles` 表
- 出资人信息通过 `profiles` 表关联查询
- 枚举类型通过映射对象转换为中文

### 性能优化
- 使用 `whitespace-nowrap` 避免文字换行
- 使用 `overflow-x-auto` 实现横向滚动
- 数字格式化使用 `toLocaleString()` 方法
- 条件渲染减少不必要的DOM元素

### 数据格式化
```typescript
// 数字格式化
vehicle.mileage.toLocaleString() // 50000 → 50,000

// 货币格式化
Number(vehicle.purchase_price).toLocaleString() // 150000 → 150,000

// 枚举转换
VEHICLE_TYPE_MAP[vehicle.vehicle_type] // sedan → 轿车
```
