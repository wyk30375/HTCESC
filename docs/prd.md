# 二手车销售管理平台需求文档

## 1. 应用概述

### 1.1 应用名称
二手车销售管理平台

### 1.2 应用描述
一款面向二手车行业的SaaS管理平台，支持多家车行同时注册使用。平台提供完整的车行管理系统功能，包括员工管理、车辆入库、销售管理、费用管理、利润分配及统计分析等。系统包含电脑端管理后台和手机端展示系统，适用于二手车行的日常运营管理和客户展示场景。\n
## 2. 核心功能模块

### 2.1 平台管理系统

- 车行注册与认证：车行可注册账号并完成身份认证后使用平台
- 车行信息管理：包含车行名称、联系方式、地址等基本信息
- 车行权限隔离：各车行数据完全隔离，互不干扰

### 2.2 员工管理系统

- 员工信息管理：包含姓名、职位、联系方式、入职时间等
- 角色设置：地租、月奖金池、销售提成、押车出资人等角色可由员工担任，支持一人多角色，多人一角色
- 员工销售业绩记录，业绩按实现利润多少评定

### 2.3 车辆入库管理

- 入库车辆信息录入：唯一标识为车架号后六位数，车牌号、品牌型号、车况、里程数、年份等
- 入库成本记录：\n  - 购车款
  - 整备费
  - 过户费
  - 杂费
- 车辆照片上传
- 车辆状态跟踪

### 2.4 车辆销售管理

- 销售信息记录：销售日期、成交价格、客户信息，销售整备费、过户费、杂费计入成本
- 车辆销售状态更新
- 销售员工关联
- 录入是否有贷款，贷款返利记录（如有）\n
### 2.5 费用管理系统

- 车行运营费用记录：包含各类日常支出
- 费用类型分类
- 费用发生时间记录

### 2.6 利润分配计算

- 总利润计算：成交价 - 总成本（购车款+整备费+过户费+杂费）+贷款返利
- 利润分配规则：\n  - 地租：18%\n  - 月奖金池：10%\n  - 销售提成：36%\n  - 押车出资人：36%\n- 支持单个员工或多个员工共同分配同一角色的利润
- 每月奖金池剩余金额作为激励奖金分配给月销售冠军

### 2.7 统计分析模块

- 月度销售统计：销售数量、销售额、平均售价等
- 员工销售排行：月度销售冠军、销售金额、销售数量、实现利润
- 利润分配明细：各角色分配金额、奖金池余额
- 销售趋势图表

### 2.8 手机端展示系统

- 客户端：展示在售车辆列表，包含车辆照片、价格、配置信息
- 内部通报端（员工专用）：\n  - 每日销售通报：当日销售数量、金额、完成率
  - 月度销售通报：当月销售数据、个人及团队业绩、奖金池情况

## 3. 系统特点

- 多租户架构：支持多家车行同时使用，数据完全隔离
- SaaS模式：统一平台部署，车行无需本地部署
- 数据安全：各车行数据独立存储，保障信息安全
- 电脑端与手机端数据同步，实时更新
- 支持多角色权限管理
- 提供完整的财务核算和统计分析功能
