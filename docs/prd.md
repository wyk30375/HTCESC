# 二手车销售管理平台需求文档

## 1. 应用概述

### 1.1 应用名称
二手车销售管理平台

### 1.2 应用描述
一款面向二手车行业的SaaS管理平台，支持多家车行同时注册使用。平台提供完整的车行管理系统功能，包括员工管理、车辆入库、销售管理、费用管理、利润分配及统计分析等。系统包含电脑端管理后台、手机端展示系统以及平台公共主页，适用于二手车行的日常运营管理和客户展示场景。\n
## 2. 核心功能模块

### 2.1 平台管理系统

- 车行注册流程：\n  - 注册人填写手机号码、营业执照信息
  - 系统获取注册人位置信息（区域定位）\n  - 阅读并同意免责条款
  - 提交注册申请
- **注册提醒功能：**\n  - 新车行提交注册申请后，系统自动向平台超级管理员发送审核提醒通知
  - 提醒方式：站内消息提醒或邮件提醒
  - 提醒内容包含车行名称、注册时间及审核链接
- 后台管理员审核：\n  - 审核车行注册信息
  - 通过审核后车行账号激活
  - 审核不通过需说明原因
- 车行信息管理：包含车行名称、联系方式、地址、营业执照等基本信息
- 车行权限隔离：各车行数据完全隔离，互不干扰
- 会员制度：\n  - 免期期：车商入驻后享有6个月免费期
  - 会员等级：\n    - 三级会员：在售车辆20台以内，年会费198元
    - 二级会员：在售车辆21至50台，年会费365元
    - 一级会员：在售车辆51至150台，年会费580元
    - 金牌会员：在售车辆151台以上，年会费980元
  - 会员续费提醒：会员到期前系统自动提醒车行续费
  - 会员状态管理：显示会员有效期、已使用车辆数量、会员等级
- 平台与车行对话功能：\n  - 车行可向平台发起建议与反馈
  - 平台可向车行发送提醒及反馈
  - 对话记录保存，支持历史消息查询
- 反馈管理系统：\n  - 车行反馈记录管理
  - 反馈处理状态跟踪
  - 反馈回复与关闭
  - 反馈统计分析

### 2.2 平台公共主页

- 车辆评估功能：\n  - 用户输入车辆信息（品牌型号、车况、里程数、年份等）\n  - 系统根据输入参数自动评估车辆价值
  - 展示评估结果，包括参考价格区间、市场行情分析
- 车辆搜索功能：支持按品牌、型号、价格区间、车况等条件筛选车辆
- 车行展示：展示已入驻车行列表，包含车行名称、地址、联系方式

### 2.3 员工管理系统

- 员工信息管理：包含姓名、职位、联系方式、入职时间、身份证照片等
- 角色设置：地租、月奖金池、销售提成、押车出资人等角色可由员工担任，支持一人多角色，多人一角色
- 员工销售业绩记录，业绩按实现利润多少评定

### 2.4 车辆入库管理

- 入库车辆信息录入：唯一标识为车架号后六位数，车牌号、品牌型号、车况、里程数、年份等
- 车辆类型识别：自动判断是否为电动车
- 电动车参数记录（当车辆类型为电动车时）：\n  - 电池容量（kWh）\n  - 续航里程（km）\n  - 充电功率（kW）\n  - 电机功率（kW）\n  - 能耗（kWh/100km）\n- 入库成本记录：\n  - 购车款
  - 整备费
  - 过户费
  - 杂费
- 车辆照片上传
- 车辆状态跟踪

### 2.5 车辆销售管理

- 销售信息记录：销售日期、成交价格、客户信息，销售整备费、过户费、杂费计入成本
- 车辆销售状态更新
- 销售员工关联
- 录入是否有贷款，贷款返利记录（如有）\n
### 2.6 费用管理系统

- 车行运营费用记录：包含各类日常支出
- 费用类型分类
- 费用发生时间记录
- 费用管理卡片快捷入口

### 2.7 利润分配计算

- 总利润计算：成交价 - 总成本（购车款+整备费+过户费+杂费）+贷款返利
- 利润分配规则：\n  - 地租：18%\n  - 月奖金池：10%\n  - 销售提成：36%\n  - 押车出资人：36%\n- 支持单个员工或多个员工共同分配同一角色的利润
- 每月奖金池剩余金额作为激励奖金分配给月销售冠军

### 2.8 统计分析模块

- 月度销售统计：销售数量、销售额、平均售价等
- 员工销售排行：月度销售冠军、销售金额、销售数量、实现利润
- 利润分配明细：各角色分配金额、奖金池情况
- 销售趋势图表

### 2.9 手机端展示系统

- 客户端：展示在售车辆列表，包含车辆照片、价格、配置信息
- 内部通报端（员工专用）：\n  - 每日销售通报：当日销售数量、金额、完成率
  - 月度销售通报：当月销售数据、个人及团队业绩、奖金池情况

## 3. 系统特点

- 多租户架构：支持多家车行同时使用，数据完全隔离
- SaaS模式：统一平台部署，车行无需本地部署
- 数据安全：各车行数据独立存储，保障信息安全
- 电脑端与手机端数据同步，实时更新
- 支持多角色权限管理
- 提供完整的财务核算和统计分析功能
- 实行会员制：根据在售车辆数量划分会员等级，享有不同年费标准
- 平台公共主页提供车辆评估工具，帮助用户快速了解车辆价值
- 平台与车行双向沟通功能：支持车行向平台反馈建议，平台向车行发送提醒
- 反馈管理系统：统一处理车行反馈，跟踪处理进度，提升服务质量
- 注册审核提醒机制：新车行注册后自动提醒平台超级管理员及时审核
