# 🚀 多车行平台快速开始指南

## 欢迎使用易驰汽车销售管理平台！

系统已成功改造为多车行SaaS平台，支持无限数量的车行同时使用。

## 📋 系统访问

- **应用地址**: http://localhost:5173
- **登录页面**: http://localhost:5173/login
- **车行注册**: http://localhost:5173/register

## 🎯 快速开始

### 方式一：创建新车行（推荐）

1. 访问 http://localhost:5173/register
2. 选择"创建新车行"标签
3. 填写车行信息和管理员账号
4. 点击"创建车行并注册"
5. 系统自动登录，开始使用！

### 方式二：加入现有车行

1. 访问 http://localhost:5173/register
2. 选择"加入车行"标签
3. 输入车行代码（向车行管理员获取）
4. 填写您的账号信息
5. 系统自动登录，开始使用！

### 方式三：使用默认车行（测试）

系统已预置默认车行"易驰汽车"：
```
用户名：admin
密码：admin123
车行代码：yichi
```

## 👥 角色说明

- **Super Admin**: 平台超级管理员，管理所有车行
- **Admin**: 车行管理员，管理本车行用户和数据
- **Employee**: 车行员工，使用本车行功能

## 🔒 数据隔离

系统使用 PostgreSQL RLS 策略确保：
- ✅ 每个车行只能看到自己的数据
- ✅ 员工只能看到所属车行的数据
- ✅ 数据库层面的安全隔离
