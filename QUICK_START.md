# 快速开始指南

## 🚀 5分钟快速上手

### 第一步：启动系统

```bash
# 进入项目目录
cd /workspace/app-8u0242wc45c1

# 安装依赖（如果还没安装）
pnpm install

# 启动开发服务器
pnpm run dev
```

系统将在 `http://localhost:5173` 启动。

### 第二步：创建管理员账号

1. 打开浏览器访问 `http://localhost:5173`
2. 点击"注册"按钮
3. 输入用户名（例如：admin）和密码
4. 点击"注册"
5. **重要**：第一个注册的用户会自动成为管理员！

### 第三步：添加员工

1. 使用管理员账号登录
2. 点击左侧菜单的"员工管理"
3. 点击右上角"添加员工"按钮
4. 填写员工信息：
   - 选择关联用户（如果有其他注册用户）
   - 输入姓名、职位、联系方式
   - 选择入职日期
5. 点击"保存"
6. 点击员工列表中的"角色"按钮，为员工分配角色：
   - 地租（18%）
   - 月奖金池（10%）
   - 销售提成（36%）
   - 押车出资人（36%）

### 第四步：录入车辆

1. 点击左侧菜单的"车辆管理"
2. 点击右上角"车辆入库"按钮
3. 填写车辆信息：
   - 车架号后六位（唯一标识）
   - 车牌号
   - 品牌和型号
   - 年份和里程数
   - 购车款
   - 车况描述
4. 上传车辆照片（支持多张，自动压缩）
5. 点击"保存"

### 第五步：记录销售

1. 点击左侧菜单的"销售管理"
2. 点击右上角"记录销售"按钮
3. 填写销售信息：
   - 选择要销售的车辆
   - 选择销售日期
   - 输入成交价格
   - 填写客户信息（姓名、联系方式、身份证号）
   - 选择销售员
   - 填写销售相关费用（整备费、过户费、杂费）
   - 如有贷款返利，勾选并填写金额
4. 点击"保存"
5. 系统会自动：
   - 计算总利润
   - 更新车辆状态为"已售"
   - 记录销售成本

### 第六步：管理费用

1. 点击左侧菜单的"费用管理"
2. 点击右上角"添加费用"按钮
3. 填写费用信息：
   - 选择费用类型（租金、水电费、工资等）
   - 输入金额
   - 选择费用日期
   - 填写费用说明
4. 点击"保存"
5. 可以按月份查看费用统计

这个页面会展示所有在售车辆，包括：

- 品牌型号
- 年份和里程

## 👥 权限说明

### 员工权限
- ✅ 可以录入新车辆
- ✅ 可以创建销售记录
- ✅ 可以添加费用记录
- ✅ 可以查看所有数据
- ❌ 不能修改已有数据
- ❌ 不能删除数据
- ❌ 不能管理员工

### 管理员权限
- ✅ 拥有所有员工权限
- ✅ 可以修改所有数据
- ✅ 可以删除数据
- ✅ 可以管理员工
- ✅ 可以分配角色
- ✅ 可以管理用户权限

## 🎯 常用功能

### 查看仪表盘
点击左侧菜单的"仪表盘"，可以看到：
- 在售车辆数量
- 本月销售数量
- 本月销售额
- 本月利润
- 最近销售记录

### 查看统计分析
点击左侧菜单的"统计分析"，可以看到：
- 月度销售趋势
- 员工销售排行
- 利润分析
- 费用统计

### 管理用户角色
1. 点击左侧菜单的"用户管理"（仅管理员可见）
2. 可以查看所有注册用户
3. 可以修改用户角色（管理员/员工）

## ❓ 遇到问题？

### 登录失败
- 检查用户名和密码是否正确
- 确保已经注册账号

### 图片上传失败
- 检查图片格式（支持 JPEG, PNG, WEBP, GIF, AVIF）
- 确保文件名只包含英文字母和数字
- 图片会自动压缩至 1MB 以下

### 权限不足
- 检查当前用户角色
- 某些操作需要管理员权限
- 重新登录以刷新权限

### 数据不显示
- 检查浏览器控制台是否有错误
- 确保 Supabase 连接正常
- 刷新页面重试

## 💡 提示

1. **首个用户是管理员**：第一个注册的用户会自动成为管理员，请妥善保管账号信息。

2. **车架号唯一**：车架号后六位是车辆的唯一标识，不能重复。

3. **角色分配灵活**：一个员工可以担任多个角色，多个员工也可以共同担任一个角色。

4. **利润自动计算**：创建销售记录时，系统会自动计算总利润。

5. **月度统计**：费用管理支持按月份查看，方便财务核算。
