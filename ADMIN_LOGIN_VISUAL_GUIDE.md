## 💡 使用技巧

### 审核效率提升

1. **快速审核**：
   - 对于资料齐全的车行，可以直接点击"通过"
   - 无需每次都查看详情

2. **批量处理**：
   - 先查看所有待审核车行的详情
   - 然后集中处理（通过或拒绝）

3. **拒绝原因模板**：
   - 准备常用的拒绝原因模板
   - 复制粘贴即可，提高效率

### 常用拒绝原因模板

```
✅ 营业执照问题：
- 营业执照图片不清晰，请重新上传高清图片
- 营业执照已过期，请更新有效证件
- 经营范围不包含二手车销售业务

✅ 信息不符：
- 企业名称与申请的车行名称不一致
- 联系电话无法接通，请提供有效联系方式

✅ 资质问题：
- 缺少二手车经营许可证，请补充相关资质
```

## ⚠️ 重要提醒

1. **及时处理**：建议24小时内处理待审核申请
2. **仔细核验**：营业执照是审核的重点，务必仔细查看
3. **明确原因**：拒绝时必须填写清晰的原因
4. **电话确认**：建议在审核前电话确认联系人身份
5. **代码唯一**：车行代码创建后不可修改，审核时需确认

## 🆘 常见问题

### Q1: 找不到"车行管理"菜单？
**A:** 请确认您的账号角色是 `super_admin`。只有超级管理员可以看到此菜单。

### Q2: 审核通过后车行还是无法登录？
**A:** 检查车行状态是否为 `active`，刷新页面重试。

### Q3: 如何撤销审核？
**A:** 可以在"正常运营"或"已拒绝"标签页找到该车行，通过"编辑"功能修改状态。

### Q4: 如何查看审核历史？
**A:** 在车行详情中可以看到审核时间和审核人信息。
