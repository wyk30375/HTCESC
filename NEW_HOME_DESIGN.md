# 全新主页设计文档

## 页面概述

**页面地址**: http://localhost:5173/register

全新设计的易驰汽车平台主页，采用现代化设计风格，清晰展示注册流程和在售车辆。

## 设计亮点

### 1. 现代化视觉设计
- **渐变背景**：使用 primary 和 accent 颜色的渐变背景
- **卡片阴影**：hover 时卡片阴影和缩放效果
- **流畅动画**：图片缩放、按钮过渡等动画效果
- **响应式布局**：完美适配桌面、平板、手机

### 2. 清晰的信息架构
- **Hero 区域**：大标题 + 副标题 + 双CTA按钮
- **注册流程说明**：三步流程可视化展示
- **平台优势**：四大特色功能展示
- **车辆展示**：精选车辆网格布局
- **底部CTA**：再次强调注册

### 3. 优化的注册体验
- **分步表单**：三步完成注册，降低认知负担
- **进度指示器**：清晰显示当前步骤
- **实时验证**：每步验证，及时反馈
- **视觉引导**：图标、颜色、文字多重引导

## 页面结构

### Header（顶部导航栏）
- **Logo区域**：图标 + 名称 + 副标题
- **未登录状态**：登录按钮 + 注册车行按钮
- **已登录状态**：车行Badge + 进入系统按钮

### Hero Section（主视觉区）
- **渐变背景**：from-primary/10 via-background to-accent/10
- **Badge标签**：专业二手车交易平台
- **主标题**：优质二手车 一站式交易平台
- **副标题**：平台价值主张
- **双CTA**：立即注册车行 + 浏览在售车辆

### How It Works（注册流程说明）
- **标题区域**：如何注册使用 + 副标题
- **三步流程卡片**：
  1. 准备资料（营业执照、基本信息、手机号）
  2. 提交审核（填写表单、上传执照、同意协议）
  3. 开始使用（等待审核、登录系统、发布车辆）
- **图标 + 列表**：清晰的步骤说明

### Platform Features（平台优势）
- **四大特色**：
  1. 安全可靠（严格审核机制）
  2. 操作简单（直观管理界面）
  3. 客户触达（统一展示平台）
  4. 数据分析（统计分析功能）
- **图标 + 标题 + 描述**：简洁明了

### Featured Vehicles（精选车辆）
- **搜索和筛选**：搜索框 + 车行筛选器
- **网格布局**：4列（xl）、3列（lg）、2列（md）、1列（sm）
- **车辆卡片**：
  - 车辆照片（hover缩放效果）
  - 品牌型号
  - 年份和里程数
  - 售价（大号字体，primary颜色）
  - 车况描述（最多2行）
  - 所属车行
- **加载状态**：骨架屏
- **空状态**：友好提示

### CTA Section（行动号召）
- **大标题**：准备好开始了吗？
- **描述文字**：价值主张
- **CTA按钮**：立即注册车行

### Footer（页脚）
- **三列布局**：
  1. 关于我们（Logo + 简介）
  2. 联系我们（电话、邮箱、地址）
  3. 快速链接（关于、协议、隐私、帮助）
- **版权信息**：© 2026 易驰汽车平台

## 注册对话框设计

### 分步表单（3步）

#### 步骤1：基本信息
- **车行名称** *（必填）
- **车行代码** *（必填，只能包含字母、数字和下划线）
- **联系人**（选填）
- **联系电话** *（必填，手机号格式验证）
- **省份** *（必填，下拉选择）
- **城市** *（必填，根据省份动态加载）
- **区县**（选填）
- **详细地址**（选填）
- **操作**：下一步按钮

#### 步骤2：资质上传
- **营业执照** *（必填）
  - 拖拽上传或点击上传
  - 支持 JPG、PNG、WEBP
  - 自动压缩至 1MB 以下
  - 上传进度条
  - 预览图片
  - 重新上传按钮
- **管理员用户名** *（必填）
- **管理员手机号** *（必填，手机号格式验证）
- **登录密码** *（必填，至少6位）
- **确认密码** *（必填，与密码一致）
- **操作**：上一步 + 下一步按钮

#### 步骤3：确认提交
- **提示信息**：请仔细阅读服务协议
- **免责条款**：滚动区域显示完整条款
- **同意勾选框** *（必填）
- **操作**：上一步 + 提交注册申请按钮

### 进度指示器
- **步骤1**：圆形数字 "1" → 完成后显示 ✓
- **步骤2**：圆形数字 "2" → 完成后显示 ✓
- **步骤3**：圆形数字 "3"
- **分隔线**：连接各步骤
- **颜色**：当前步骤 primary，未完成 muted

### 验证逻辑
- **步骤1验证**：
  - 车行名称、代码、联系电话、省份、城市必填
  - 车行代码格式验证
  - 手机号格式验证
- **步骤2验证**：
  - 营业执照、用户名、手机号、密码必填
  - 手机号格式验证
  - 密码长度验证（至少6位）
  - 密码一致性验证
- **步骤3验证**：
  - 必须勾选同意协议

### 提交流程
1. 创建车行（status = pending）
2. 创建管理员账号
3. 更新用户资料（role = admin）
4. 显示成功提示
5. 关闭对话框
6. 重置表单
7. 不自动登录（需等待审核）

## 技术实现

### 组件和工具
- **UI组件**：Card, Button, Input, Label, Dialog, Tabs, Badge, Select, Checkbox, Progress, Separator, Alert, Skeleton
- **图标**：Lucide React（30+ 图标）
- **图片上传**：uploadBusinessLicense（自动压缩）
- **区域数据**：PROVINCES, getCitiesByProvince
- **免责条款**：DisclaimerContent

### 数据获取
- **车行列表**：status = active，按创建时间倒序
- **车辆列表**：status = in_stock，关联车行信息，限制12辆，按创建时间倒序

### 响应式设计
- **容器**：container 类，自动适配屏幕宽度
- **网格布局**：grid-cols-1 md:grid-cols-2 lg:grid-cols-3 xl:grid-cols-4
- **弹性布局**：flex flex-col sm:flex-row
- **间距**：gap-3 md:gap-6 lg:gap-8
- **字体大小**：text-4xl md:text-6xl

### 动画效果
- **图片缩放**：group-hover:scale-105 transition-transform duration-300
- **卡片阴影**：hover:shadow-lg transition-all duration-300
- **按钮过渡**：transition-colors
- **平滑滚动**：scrollIntoView({ behavior: 'smooth' })

## 设计原则

### 1. 用户体验优先
- **降低认知负担**：分步表单，每步只关注少量信息
- **及时反馈**：实时验证，错误提示
- **视觉引导**：图标、颜色、文字多重引导
- **流畅交互**：动画过渡，平滑滚动

### 2. 信息清晰
- **层次分明**：标题、副标题、正文、辅助文字
- **重点突出**：CTA按钮、价格、状态标签
- **留白充足**：间距合理，不拥挤

### 3. 视觉美观
- **现代风格**：渐变、阴影、圆角
- **色彩和谐**：使用设计系统颜色
- **图标丰富**：每个功能都有对应图标

### 4. 性能优化
- **图片懒加载**：loading="lazy"
- **骨架屏**：加载状态友好
- **限制数量**：首页只显示12辆车

## 使用建议

### 访客用户
1. 浏览Hero区域了解平台
2. 查看注册流程说明
3. 浏览精选车辆
4. 点击"立即注册车行"开始注册

### 车行注册
1. 准备好营业执照扫描件
2. 点击"注册车行"按钮
3. 按步骤填写信息
4. 上传营业执照
5. 阅读并同意协议
6. 提交等待审核

### 已登录用户
1. 看到"进入系统"按钮
2. 可以浏览车辆
3. 点击进入管理后台
