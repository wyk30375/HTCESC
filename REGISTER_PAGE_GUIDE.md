# 车行注册页面使用说明

## 页面地址
http://localhost:5173/register

## 功能说明

车行注册页面支持两种注册方式：

### 1. 创建新车行
适用于首次使用平台的车行管理员。

**步骤：**
1. 访问注册页面
2. 选择"创建新车行"标签
3. 填写车行信息：
   - 车行名称（必填）
   - 车行代码（必填，用于员工加入）
   - 联系人（可选）
   - 联系电话（可选）
   - 车行地址（可选）
4. 填写管理员账号：
   - 用户名（必填）
   - 密码（必填，至少6位）
   - 确认密码（必填）
   - 手机号（可选）
5. 点击"创建车行并注册"
6. 系统自动登录，跳转到仪表盘

### 2. 加入现有车行
适用于加入已有车行的员工。

**步骤：**
1. 访问注册页面
2. 选择"加入车行"标签
3. 输入车行代码（向车行管理员获取）
4. 填写员工账号：
   - 用户名（必填）
   - 密码（必填，至少6位）
   - 确认密码（必填）
   - 手机号（可选）
5. 点击"加入车行并注册"
6. 系统自动登录，跳转到仪表盘

## 已登录用户访问

如果您已经登录，访问注册页面时会看到：

- **提示信息**：显示当前登录的账号和车行
- **返回首页**：直接返回系统首页
- **退出登录后注册新车行**：退出当前账号，刷新页面后可以注册新车行

**注意**：已登录用户无法直接注册新车行，需要先退出登录。

## 常见问题

**Q: 车行代码有什么要求？**
A: 只能包含字母、数字和下划线，全局唯一，创建后不可修改。

**Q: 忘记车行代码怎么办？**
A: 联系车行管理员获取，或在数据库中查询。

**Q: 用户名重复怎么办？**
A: 用户名在整个平台上必须唯一，请更换其他用户名。

**Q: 如何测试多车行功能？**
A: 参考 TESTING_GUIDE.md 文档中的详细测试步骤。
