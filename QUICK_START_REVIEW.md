# 🚀 车行审核功能快速开始

## 📌 核心流程（3步完成审核）

### 第1步：登录管理后台
```
访问：http://localhost:5173/login
账号：吴韩（超级管理员）
密码：您的密码
```

### 第2步：进入车行管理
```
登录后 → 左侧导航栏 → 点击"车行管理"
```

### 第3步：审核车行
```
✅ 通过：查看详情 → 确认营业执照 → 点击"通过"按钮
❌ 拒绝：点击"拒绝"按钮 → 填写拒绝原因 → 确认拒绝
```

---

## 🎯 审核要点（3个关键检查）

### ✅ 1. 营业执照
- 图片清晰度
- 企业名称一致性
- 经营范围包含二手车销售
- 有效期内

### ✅ 2. 联系方式
- 联系人姓名真实
- 联系电话可接通
- 建议电话确认身份

### ✅ 3. 车行代码
- 简洁易记
- 只包含字母、数字、下划线
- 创建后不可修改

---

## 📊 页面功能说明

### 四个标签页

| 标签 | 说明 | 操作 |
|------|------|------|
| 🔴 **待审核** | 新提交的申请（显示数量徽章） | 查看、通过、拒绝 |
| ✅ **正常运营** | 已审核通过的车行 | 查看、编辑、停用 |
| ⏸️ **已停用** | 被临时停用的车行 | 查看、编辑、启用 |
| ❌ **已拒绝** | 审核未通过的申请 | 查看详情 |

---

## 🔐 登录限制说明

审核通过后，车行才能登录使用系统：

| 车行状态 | 登录结果 | 提示信息 |
|---------|---------|---------|
| ⏳ **pending** | ❌ 拒绝登录 | "您的车行正在审核中，请等待管理员审核通过后再登录" |
| ✅ **active** | ✅ 允许登录 | 正常使用所有功能 |
| ⏸️ **inactive** | ❌ 拒绝登录 | "您的车行已被停用，请联系管理员" |
| ❌ **rejected** | ❌ 拒绝登录 | "您的车行注册申请已被拒绝，请联系管理员了解详情" |

**注意：** 超级管理员不受车行状态限制，始终可以登录。

## 💡 常用操作

### 审核通过（推荐流程）
```
1. 点击"查看"按钮 👁️
2. 核对营业执照图片 📄
3. 确认联系方式 📞
4. 关闭详情对话框
5. 点击"通过"按钮 ✅
6. 完成！车行可以登录了 🎉
```

### 审核拒绝（需填写原因）
```
1. 点击"拒绝"按钮 ❌
2. 填写拒绝原因（必填）
   例如："营业执照图片不清晰，请重新上传"
3. 点击"确认拒绝"
4. 完成！申请人会看到拒绝原因
```

### 临时停用车行
```
1. 切换到"正常运营"标签页
2. 找到要停用的车行
3. 点击红色电源图标 🔴
4. 车行立即无法登录
```

### 重新启用车行
```
1. 切换到"已停用"标签页
2. 找到要启用的车行
3. 点击绿色电源图标 🟢
4. 车行恢复正常使用
```

## 📝 拒绝原因示例

复制粘贴即可使用：

```
✅ 营业执照问题：
- "营业执照图片不清晰，请重新上传高清图片"
- "营业执照已过期，请更新有效证件"
- "经营范围不包含二手车销售业务"

✅ 信息不符：
- "企业名称与申请的车行名称不一致"
- "联系电话无法接通，请提供有效联系方式"

✅ 资质问题：
- "缺少二手车经营许可证，请补充相关资质"
- "企业已注销，无法通过审核"
```

## ⚠️ 重要提醒

1. **及时处理**：建议24小时内处理待审核申请
2. **仔细核验**：营业执照是审核的重点，务必仔细查看
3. **明确原因**：拒绝时必须填写清晰的原因
4. **电话确认**：建议在审核前电话确认联系人身份
5. **代码唯一**：车行代码创建后不可修改，审核时需确认

## 🆘 遇到问题？

### 找不到"车行管理"菜单？
→ 确认您的账号是超级管理员（super_admin）

### 审核通过后车行还是无法登录？
→ 检查车行状态是否为 active，刷新页面重试

### 如何查看审核历史？
→ 在车行详情中可以看到审核时间和审核人
